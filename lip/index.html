• LIPTON ICE TEA: A-Z + FRIEND SOAKER
    2010 - 2011

    ANIMATION + DEV

  • The A-Z concept was to showcase 26 different ways to cool down, which meant 26 different animated scenes, compressed into a rich flash banner ad for Lipton Ice Tea.

  • Great animation exercise with the time allocated. The interactive flash banner was also adapted into a desktop screensaver.

  • Friend Soaker was a fun water-bombing interactive flash game between Facebook friends. Animation of the characters were set up with varying speeds. Via Soap Creative.
